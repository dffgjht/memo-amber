# 🎯 如何获取 APK 安装包

## ⚡ 最快方法（推荐）

### 使用 Android Studio（仅 20 分钟）

```
1. 下载: https://developer.android.com/studio
2. 安装（全部默认设置）
3. 打开项目文件夹
4. Build → Build APK(s)
5. ✅ 完成！
```

**APK 位置**: `app/build/outputs/apk/debug/app-debug.apk`

---

## ☁️ 云端构建（无需安装工具）

### 使用 GitHub Actions

```
1. 将项目推送到 GitHub
2. 进入 Actions 页面
3. 点击 "Run workflow"
4. 等待 5-10 分钟
5. 下载自动生成的 APK
```

**详细步骤**: 见 `INSTALLATION_GUIDE.md`
