# 📔 Death Diary - 数字化死亡日记

一款专注于个人反思、情感表达和生命记录的私密应用。记录你的日常、珍藏回忆、编写遗嘱，让每一段时光都有迹可循。

## ✨ 核心功能

- **📔 日记模块** - 轻松记录生活点滴，支持位置标记、照片预览
- **🔐 密码保险箱** - 安全存储重要信息，加密保护隐私
- **📜 遗嘱遗言** - 提前规划身后事宜，自动发送重要信息
- **📷 回忆相册** - 珍藏美好时刻，智能分类管理
- **⚙️ 设置优化** - 移除不稳定的生物识别功能

## 🚀 特性亮点

- **隐私保护** - 所有数据本地存储，端到端加密
- **离线可用** - 无网络连接也能正常使用
- **多平台支持** - Android 8.0+ 完美运行
- **智能提醒** - 自动发送遗嘱信息给指定联系人

## 📦 安装说明

1. 下载最新 APK 安装包
2. 启用 USB 调试模式（如需从电脑安装）
3. 在安装页面选择"允许"
4. 首次启动需授予存储权限

> ⚠️ 安全提示：请谨慎共享您的账户信息，应用支持本地加密备份功能
