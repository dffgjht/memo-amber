# 死亡日记 (Death Diary) - 数字遗产系统

## 项目概述
一个安卓应用，用于记录和保存生前的重要数据、记忆、遗言等数字遗产。

## 核心功能
1. **日记记录** - 文字、图片、语音日记
2. **重要信息存储** - 加密保存账号、密码、重要文件
3. **遗言/遗嘱** - 数字遗嘱功能
4. **回忆相册** - 珍贵照片和视频
5. **定时发送** - 设定在特定时间或条件下发送给指定联系人
6. **数据导出** - 导出为加密的备份文件

## 技术栈
- Kotlin + Jetpack Compose (现代化UI)
- Room Database (本地存储)
- Encryption (AES加密敏感数据)
- Coroutines + Flow (异步处理)
- Material Design 3 (UI设计)

## 安全考虑
- 所有敏感数据AES-256加密
- 生物识别认证（指纹/面容）
- 主密码保护
- 自动锁定机制
- 加密备份
